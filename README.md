# SOFTYS INCORP Website

Deploy-ready Next.js website for SOFTYS INCORP.

## Local run
```bash
npm install
npm run dev
```

## Production build
```bash
npm install
npm run build
npm run start
```

## Deploy to Vercel
1. Sign in to Vercel.
2. Import this folder or upload the zip contents to GitHub.
3. Vercel will detect Next.js automatically.
4. Click Deploy.

## Domain
After deploy, add:
- softys.in
- www.softys.in

Then update DNS records at your domain provider according to Vercel's instructions.
